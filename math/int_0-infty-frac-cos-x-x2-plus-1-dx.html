Integrate \( \int_0^\infty \frac{\cos x}{x^2+1} \; dx \)

\[ I = \int_0^\infty \frac{\cos x}{x^2+1} \; dx \] Another form of this integral, which can be easily seen using elementary substitutions is: \[ J = \int_0^{\frac{\pi}{2}}\cos (\tan x) \; dx \] In order to use Feynman's method of differentiating under the sign of the integral, we need to parametrize the integral as: \[ I(a) = \int_0^\infty \frac{\cos ax}{x^2+1} \; dx \tag{1} \] Differentiating both sides w.r.t a, we get: \[ \frac{dI}{da} = \frac{d}{da} \; \int_0^\infty \frac{\cos ax}{x^2+1} \; dx = \int_0^\infty \frac{\partial}{\partial a} \frac{\cos ax}{x^2+1} \; dx \] \[ \frac{dI}{da} = \int_0^\infty \frac{-\sin ax}{x^2+1} x \; dx = - \int_0^\infty \frac{x \sin ax}{x^2+1} \; dx \] Rearranging the above integral, we have: \[ \frac{dI}{da} = - \int_0^\infty \frac{x^2}{x^2+1} \frac{\sin ax}{x} \; dx = - \int_0^\infty \frac{x^2 + 1 - 1}{x^2+1} \frac{\sin ax}{x} \; dx \] \[ \frac{dI}{da} = - \int_0^\infty \frac{\sin ax}{x} \; dx + \int_0^\infty \frac{1}{x^2+1} \frac{\sin ax}{x} \; dx \] Using the result from the Dirichlet integral, we know that: \[ \int_0^\infty \frac{\sin ax}{x} \; dx = \frac{\pi}{2} \] Thus, \[ \frac{dI}{da} = - \frac{\pi}{2} + \int_0^\infty \frac{\sin ax}{x(x^2+1)} \; dx \tag{2} \] Differentiating both sides w.r.t a again, we get: \[ \frac{d^2I}{da^2} = \frac{d}{da} \; (- \frac{\pi}{2} + \int_0^\infty \frac{\sin ax}{x(x^2+1)} \; dx) = 0 + \frac{d}{da} \int_0^\infty \frac{\sin ax}{x(x^2+1)} \; dx \] \[ \frac{d^2I}{da^2} = \int_0^\infty \frac{\partial}{\partial a} \frac{\sin ax}{x(x^2+1)} \; dx \] \[ \frac{d^2I}{da^2} = \int_0^\infty \frac{\cos ax}{x(x^2+1)} x \; dx \] \[ \frac{d^2I}{da^2} = \int_0^\infty \frac{\cos ax}{x^2+1} \; dx \] From equation (1), we have: \[ \frac{d^2I}{da^2} = I \tag{3} \] The above is a second order linear differential equation with the following boundary conditions: \[ I(0) = \int_0^\infty \frac{\cos 0}{x^2+1} \; dx = \int_0^\infty \frac{1}{x^2+1} \; dx = \tan^{-1} x \Biggr|_0^\infty \] \[ I(0) = \frac{\pi}{2} \tag{4} \] From equation (2), we have \[ I^{\prime}(0) = - \frac{\pi}{2} \tag{5} \] The characteristic equation for (3) is: \[ \lambda^2 - 1 = 0 \] The solutions are: \[ \lambda_1 = 1, \; \lambda_2 = -1 \] Hence the solution of equation (3) is of the form: \[ I(a) = Ae^{{\lambda_1}a} + Be^{{\lambda_1}a} = Ae^{a} + Be^{-a} \] \[ I^{\prime}(a) = Ae^{a} - Be^{-a} \] Using the boundary conditions from equations (4) and (5), we have: \[ I(0) = \frac{\pi}{2} = Ae^{0} + Be^{-0} = A+B \] \[ I^{\prime}(0) = -\frac{\pi}{2} = Ae^{0} - Be^{-0} = A-B \] Solve for A and B: \[ A = 0, \; B = \frac{\pi}{2} \] Hence we have: \[ I(a) = \frac{\pi}{2} e^{-a} \] The solution of the original integral is when \( a = 1 \). Thus: \[ I(1) = \int_0^\infty \frac{\cos x}{x^2+1} \; dx = \frac{\pi}{2} e^{-1} \] \[ \int_0^\infty \frac{\cos x}{x^2+1} \; dx = \frac{\pi}{2e} \]
