Integrate \( \int\frac{dx}{x^3 + 1} \)

\[ I = \int\frac{dx}{x^3 + 1} \] Since \( x^3 + 1 = (x+1)(x^2 - x + 1) \),
\[ I = \int\frac{dx}{(x + 1)(x^2 - x + 1)} \] Decomposing the denominator using partial fractions gives us the following:
\[ \frac{1}{x^3 + 1} = \frac{A}{x + 1} + \frac{Bx+c}{x^2 - x + 1} \] \[ 1 = A(x^2 - x + 1) + (Bx+C)(x+1) \] \[ 1 = Ax^2 - Ax + A + Bx^2 + Bx +Cx + C \] \[ 1 = (A+B)x^2 + (-A + B + C)x + (A + C) \] This yields the following system of equations:
\[ A + B = 0 \] \[ -A + B + C = 0 \] \[ A + C = 1 \] Solving for A, B, C gives: \( A = \frac{1}{3} \), \( B = -\frac{1}{3} \), \( C = \frac{2}{3} \)
