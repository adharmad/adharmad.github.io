Integrate \( \int_0^\infty \frac{\sin^2 x}{x^2} \; dx \)

\[ I = \int_0^\infty \frac{\sin^2 x}{x^2} \; dx \] In this integral, we will use the result from \( \int_0^\infty \frac{\sin x}{x} \; dx \)

Using Feynman's technique of differentiating under the sign of the integral, let us first parametrize the integral: \[ I(a) = \int_0^\infty \frac{\sin^2 (ax)}{x^2} \; dx \] \[ \frac{dI}{da} = \frac{d}{da} \int_0^\infty \frac{\sin^2 (ax)}{x^2} \; dx = \int_0^\infty \frac{\partial}{\partial a} \frac{\sin^2 (ax)}{x^2} \; dx \] \[ \frac{dI}{da} = \int_0^\infty \frac{1}{x^2} 2 \sin (ax) \cos (ax) \; x \; dx = \int_0^\infty \frac{2 \sin (ax) \cos (ax)}{x} \; dx \] \[ \frac{dI}{da} = \int_0^\infty \frac{\sin (2ax)}{x} \; dx \] Substituting \( 2ax = t \Rightarrow dx = \frac{dt}{2a} \), the limits of the integration are unchanged: \[ \frac{dI}{da} = \int_0^\infty \frac{\sin t}{\frac{t}{2a}} \; \frac{dt}{2a} = \int_0^\infty \frac{\sin t}{t} \; dt \] Now we know from the above referenced link that: \[ \int_0^\infty \frac{\sin x}{x} \; dx = \frac{\pi}{2} \] Thus: \[ \frac{dI}{da} = \frac{\pi}{2} \] Integrating both sides w.r.t a, we get: \[ I = \int \frac{\pi}{2} \; da = \frac{\pi}{2} a + c \] From the original parametrized integral, \( I(0) = 0 \). Hence \[ I(0) = \frac{\pi}{2} (0) + c \] \[ c = 0 \] \[ I = \frac{\pi}{2} a \] The value of the integral that we set out to solve is when we set a = 1: \[ I(1) = \frac{\pi}{2} (1) \] \[ \int_0^\infty \frac{\sin^2 x}{x^2} \; dx = \frac{\pi}{2} \]
