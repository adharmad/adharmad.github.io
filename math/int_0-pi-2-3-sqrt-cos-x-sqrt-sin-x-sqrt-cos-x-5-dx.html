Integrate \( \int_0^\frac{\pi}{2} \frac{3\sqrt{\cos x}}{(\sqrt{\sin x} + \sqrt{\cos x})^5} \; dx \)

\[ I = \int_0^\frac{\pi}{2} \frac{3\sqrt{\cos x}}{(\sqrt{\sin x} + \sqrt{\cos x})^5} \; dx \tag{1} \] Using King's identity, \( I = \int_a^bf(x) \; dx = \int_a^bf(b+a-x) \; dx \), we get: \[ I = 3 \int_0^\frac{\pi}{2} \frac{\sqrt{\cos (\frac{\pi}{2} - x)}}{(\sqrt{\sin (\frac{\pi}{2} - x)} + \sqrt{\cos (\frac{\pi}{2} - x)})^5} \; dx \] \[ I = 3 \int_0^\frac{\pi}{2} \frac{\sqrt{\sin x}}{(\sqrt{\cos x} + \sqrt{\sin x})^5} \; dx \tag{2} \] Adding equations (1) and (2): \[ 2I = 3 \int_0^\frac{\pi}{2} \frac{\sqrt{\cos x}}{(\sqrt{\sin x} + \sqrt{\cos x})^5} \; dx + 3 \int_0^\frac{\pi}{2} \frac{\sqrt{\sin x}}{(\sqrt{\cos x} + \sqrt{\sin x})^5} \; dx \] \[ I = \frac{3}{2} \int_0^\frac{\pi}{2} \frac{\sqrt{\sin x} + \sqrt{\cos x}}{(\sqrt{\sin x} + \sqrt{\cos x})^5} \; dx = \frac{3}{2} \int_0^\frac{\pi}{2} \frac{1}{(\sqrt{\sin x} + \sqrt{\cos x})^4} \; dx \] \[ I = \frac{3}{2} \int_0^\frac{\pi}{2} \frac{1}{(\sqrt{\cos x} (\sqrt{\tan x} + 1))^4} \; dx = \frac{3}{2} \int_0^\frac{\pi}{2} \frac{1}{\cos^2 x (\sqrt{\tan x} + 1)^4} \; dx \] \[ I = \frac{3}{2} \int_0^\frac{\pi}{2} \frac{\sec^2 x}{(\sqrt{\tan x} + 1)^4} \; dx \] Substituting \( tan x = t \Rightarrow \sec^2 x \; dx = dt \), the limits of integration go from \( x=0, x=\frac{\pi}{2} \) to \( t = 0, t = \infty \): \[ I = \frac{3}{2} \int_0^\infty \frac{1}{(\sqrt{t} + 1)^4} \; dt \] Substituting \( t = u^2 \Rightarrow dt = 2u \; du \), the limits of integration go from \( t=0, t=\infty \) to \( u = 0, u = \infty \): \[ I = \frac{3}{2} \int_0^\infty \frac{1}{(u + 1)^4} 2u \; du \] \[ I = 3 \int_0^\infty \frac{u}{(u + 1)^4} \; du \] This integral can be solved in two ways:

Method 1: Algebraic manipulations
Substituting \( u+1 = v \Rightarrow du = dv \), the limits of integration go from \( u=0, u=\infty \) to \( v = 1, v = \infty \): \[ I = 3 \int_1^\infty \frac{v-1}{v^4} \; dv = 3 \int_1^\infty (v^{-3} - v^{-4}) \; dv \] \[ I = 3 \; (-\frac{1}{2v^2} + \frac{1}{3v^3}) \Biggr|_1^\infty = 3 \; \Biggl( (-0 + 0) - (-\frac{1}{2} + \frac{1}{3}) \Biggr) = 3 (\frac{1}{2} - \frac{1}{3}) \] \[ I = \frac{1}{2} \]

Method 2: Special functions
Using the definition of the beta function, we have: \[ B(p, q) = \int_0^\infty \frac{t^{p-1}}{(1+t)^{p+q}} \; dt \] We can write the integral as: \[ I = 3 \int_0^\infty \frac{u^{2 - 1}}{(1+u)^{2+2}} \; dt \] \[ I = B(2, 2) \] Using the relation between the beta and gamma function: \[ B(p, q) = \frac{\Gamma(p) \Gamma(q)}{\Gamma(p+q)} \] Thus, \[ I = 3 \frac{\Gamma(2) \Gamma(2)}{\Gamma(2+2)} = 3 \frac{\Gamma(2) \Gamma(2)}{\Gamma(4)} = 3 \frac{1! 1! }{3!} = \frac{3}{3!} \] \[ I = \frac{1}{2} \]
