\[ I = \int \sqrt{\tan x} \; dx \] Substitute \( \tan x = t^2\). \[ \frac{t^2}{(t^2 + t\sqrt{2} + 1)(t^2 - t\sqrt{2} + 1)} = \frac{At+B}{(t^2 + t\sqrt{2} + 1)} + \frac{Ct+D}{(t^2 - t\sqrt{2} + 1)} \] \[ t^2 = (At+B)(t^2 - t\sqrt{2} + 1) + (Ct+D)(t^2 + t\sqrt{2} + 1) \] \[ t^2 = At^3 - A\sqrt{2}t^2 + At + Bt^2 - B\sqrt{2}t + B + Ct^3 + C\sqrt{2}t^2 + Ct + Dt^2 + D\sqrt{2}t + D \] \[ t^2 = (A + C)t^3 + (-A\sqrt{2} + B + C\sqrt{2} + D)t^2 + (A - B\sqrt{2}t + C + D\sqrt{2})t + (B+D) \] This yields the following system of equations:
\[ A + C = 0 \] \[ -A\sqrt{2} + B + C\sqrt{2} + D = 1 \] \[ A - B\sqrt{2}t + C + D\sqrt{2} = 0 \] \[ B + D = 0 \] Solving for A, B, C, D gives: \( A = -\frac{1}{2\sqrt{2}} \), \( B = 0 \), \( C = \frac{1}{2\sqrt{2}} \), \( D = 0 \)
