Integration Problem 3

The Gaussian Integral, also called the Euler-Poisson Integral:

\[ I = \int_{-\infty}^\infty e^{-x^2} dx \tag{1} \] We can change the variable of integration and write the same integral as
\[ I = \int_{-\infty}^\infty e^{-y^2} dy \tag{2} \] Multiplying equations (1) and (2):
\[ I^2 = \int_{-\infty}^\infty e^{-x^2} dx \times \int_{-\infty}^\infty e^{-y^2} dy \] Rearranging the expression:
\[ I^2 = \int_{-\infty}^\infty \int_{-\infty}^\infty e^{-x^2} e^{-y^2} dx dy \] \[ I^2 = \int_{-\infty}^\infty \int_{-\infty}^\infty e^{-(x^2 + y^2)} dx dy \] Changing from carterian coordinates to polar coordinates we get:
\[ x = r \cos \theta , y = r \sin \theta \] \[ x^2 + y^2 = r^2, \theta = \tan^{-1}({\frac{y}{x}}) \] and
\[ dx dy = r dr d\theta \] The limits of integration change from \( -\infty < x < \infty, -\infty < y < \infty \) to \( 0 < r < 0, 0 < \theta < 2\pi \) Therefore
\[ I^2 = \int_0^\infty \int_0^{2\pi} e^{-r^2} r dr d\theta = \int_0^{2\pi} d\theta \times \int_0^\infty e^{-r^2} r dr \] \[ I^2 = \Bigr[ \theta \Bigr]_0^{2\pi} \times \int_0^\infty r e^{-r^2} r dr = 2\pi \int_0^\infty r e^{-r^2} dr \] Substituting \( t = r^2, dt = 2 r dr, r dr = \frac{dt}{2} \), we get: \[ I^2 = = 2\pi \int_0^\infty e^{-t} \frac{dt}{2} = \pi \int_0^\infty e^{-t} dt = \pi \Bigr[ -e^{-t} \Bigr]_0^\infty = \pi [0 - (-1)] \] \[ I^2 = \pi \] \[ I = \sqrt\pi \]
