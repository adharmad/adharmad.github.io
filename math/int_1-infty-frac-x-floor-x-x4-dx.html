Integrate \( \int_1^\infty \frac{x-\lfloor x \rfloor}{x^4} \; dx \)

\[ I = \int_1^\infty \frac{x-\lfloor x \rfloor}{x^4} \; dx \] \[ I = \int_1^\infty (\frac{x}{x^4} - \frac{\lfloor x \rfloor}{x^4}) \; dx = \int_1^\infty (\frac{1}{x^3} - \frac{\lfloor x \rfloor}{x^4}) \; dx \] \[ I = \int_1^\infty \frac{1}{x^3} \; dx - \int_1^\infty \frac{\lfloor x \rfloor}{x^4} \; dx \] \[ I = -\frac{1}{2x^2} \Biggr|_1^\infty - \int_1^\infty \frac{\lfloor x \rfloor}{x^4}) \; dx = - (0 - \frac{1}{2}) - \int_1^\infty \frac{\lfloor x \rfloor}{x^4} \; dx \] \[ I = \frac{1}{2} - \int_1^\infty \frac{\lfloor x \rfloor}{x^4} \; dx \tag{1} \] Let us evaluate \( \int_1^\infty \frac{\lfloor x \rfloor}{x^4} \; dx \) separately. Say: \[ J = \int_1^\infty \frac{\lfloor x \rfloor}{x^4} \; dx \] Since the numerator is a floor function which will have different fixed values in the various sub-intervals of the integral, we can re-write the integral as: \[ J = \int_1^2 \frac{\lfloor x \rfloor}{x^4} \; dx + \int_2^3 \frac{\lfloor x \rfloor}{x^4} \; dx + \int_3^4 \frac{\lfloor x \rfloor}{x^4} \; dx + \int_4^5 \frac{\lfloor x \rfloor}{x^4} \; dx + ... \] Now, we can use the values of the floor function in the various intervals: \[ \lfloor x \rfloor = 1 \; for \; x \; in \; (1, 2) \] \[ \lfloor x \rfloor = 2 \; for \; x \; in \; (2, 3) \] \[ \lfloor x \rfloor = 3 \; for \; x \; in \; (3, 4) \] \[ \lfloor x \rfloor = 4 \; for \; x \; in \; (4, 5) \] and so on. Thus, \[ J = \int_1^2 \frac{1}{x^4} \; dx + \int_2^3 \frac{2}{x^4} \; dx + \int_3^4 \frac{3}{x^4} \; dx + \int_4^5 \frac{4}{x^4} \; dx + ... \] \[ J = - \frac{1}{3x^3} \Biggr|_1^2 - 2 \frac{1}{3x^3} \Biggr|_2^3 - 3 \frac{1}{3x^3} \Biggr|_3^4 - 4 \frac{1}{3x^3} \Biggr|_4^5 - ... \] \[ J = -\frac{1}{3} \Biggl( (\frac{1}{2^3} - 1) + 2 (\frac{1}{3^3} - \frac{1}{2^3}) + 3 (\frac{1}{4^3} - \frac{1}{3^3}) + 4 (\frac{1}{5^3} - \frac{1}{4^3}) - ... \Biggr) \] \[ J = -\frac{1}{3} \Biggl( -1 + (1-2) \frac{1}{2^3} + (2-3) \frac{1}{3^3} + (3-4) \frac{1}{4^3} + (4-5) \frac{1}{5^3} + ... \Biggr) \] \[ J = \frac{1}{3} \Biggl( 1 + \frac{1}{2^3} + \frac{1}{3^3} + \frac{1}{4^3} + \frac{1}{5^3} + ... \Biggr) = \frac{1}{3} \sum_{k=1}^\infty \frac{1}{k^3} \] The expression in the above integral is none other than Ap√©ry's constant, or the Riemann zeta function of 3. \[ J = \frac{1}{3} \zeta{(3)} \] From equation (1), we know that: \[ I = \frac{1}{2} - J \] \[ I = \frac{1}{2} - \frac{1}{3} \zeta{(3)} \]
