Integrate \( \int \frac{x^4 + 1}{x^6 + 1} \; dx \)

\[ I = \int \frac{x^4 + 1}{x^6 + 1} \; dx \] Since \( x^6 + 1 = (x^2)^3 + 1 = (x^2+1)(x^4 - x^2 + 1) \),
\[ I = \int \frac{x^4 + 1}{(x^2+1)(x^4 - x^2 + 1)} \; dx \] \[ I = \int \frac{x^4 - x^2 + 1 + x^2 }{(x^2+1)(x^4 - x^2 + 1)} \; dx = \int \frac{dx}{x^2+1} + \int \frac{x^2}{(x^2+1)(x^4 - x^2 + 1)} \; dx \] \[ I = \int \frac{dx}{x^2+1} + I_1 \] \[ I = \tan^{-1}x + I_1 \tag{1} \] where \[ I_1 = \int \frac{x^2}{(x^2+1)(x^4 - x^2 + 1)} \; dx \tag{2} \] Consider the fraction in \(I_1\) : \(\frac{x^2}{(x^2+1)(x^4 - x^2 + 1)} \). Temporarily using \( x^2 = t \) and decomposing the denominator using partial fractions gives us the following:
\[ \frac{x^2}{(x^2+1)(x^4 - x^2 + 1)} = \frac{t}{(t+1)(t^2 - t + 1)} = \frac{A}{t + 1} + \frac{Bt+c}{t^2 - t + 1} \] \[ t = A(t^2 -t + 1) + (Bt+C)(t+1) \] \[ t = At^2 - At + A + Bt^2 + Bt +Ct+C \] \[ t = (A+B) t^2 + (-A + B + C)t + (A + C) \] This yields the following system of equations:
\[ A + B = 0 \] \[ - A + B + C = 1 \] \[ A + C = 0 \] Solving for A, B, C gives: \( A = -\frac{1}{3} \), \( B = \frac{1}{3} \), \( C = \frac{1}{3} \)
