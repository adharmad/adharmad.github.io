\[ I = \int\frac{dx}{x^4 + 1} \] Since \( x^4 + 1 = x^4 + 2x^2 + 1 - 2x^2 = (x^2 + 1)^2 - (x\sqrt{2})^2 = (x^2 + x\sqrt{2} + 1)(x^2 - x\sqrt{2} + 1) \),
\[ I = \int\frac{dx}{(x^2 + x\sqrt{2} + 1)(x^2 - x\sqrt{2} + 1)} \] Decomposing the denominator using partial fractions gives us the following:
\[ \frac{1}{(x^2 + x\sqrt{2} + 1)(x^2 - x\sqrt{2} + 1)} = \frac{Ax+B}{x^2 + x\sqrt{2} + 1} + \frac{Cx+D}{x^2 - x\sqrt{2} + 1} \] \[ 1 = (Ax+B)(x^2 - x\sqrt{2} + 1) + (Cx+D)(x^2 + x\sqrt{2} + 1) \] \[ 1 = (A+C)x^3 + (-A\sqrt{2} + B + C\sqrt{2} + D)x^2 + (A - B\sqrt{2} + C + D\sqrt{2})x + (B+D) \] This yields the following system of equations:
\[ A + C = 0 \] \[ -A\sqrt{2} + B + C\sqrt{2} + D = 0 \] \[ A - B\sqrt{2} + C + D\sqrt{2} = 0 \] \[ B + D = 1 \] Solving for A, B, C, D gives: \( A = \frac{1}{2\sqrt{2}} \), \( B = \frac{1}{2} \), \( C = -\frac{1}{2\sqrt{2}} \), \( D = \frac{1}{2} \)
