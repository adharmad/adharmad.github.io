Integrate \( \int_0^\infty \frac{x}{e^x - 1} \; dx \)

\[ I = \int_0^\infty \frac{x}{e^x - 1} \; dx \] \[ I = \int_0^\infty \frac{x e^{-x}}{1 - e^{-x}} \; dx \] We know from the well known geometric series that: \[ \frac{1}{1-x} = 1 + x + x^2 + x^3 +... \] Thus, \[ \frac{1}{1-e^{-x}} = 1 + e^{-x} + e^{-2x} + e^{-3x} +... = \sum_{n=0}^\infty e^{-nx} \] Hence, the integral becomes" \[ I = \int_0^\infty x e^{-x} \sum_{n=0}^\infty e^{-nx} \; dx = \int_0^\infty x \sum_{n=0}^\infty e^{-(n+1)x} \; dx \] Interchanging the integral and summation, \[ I = \sum_{n=0}^\infty \int_0^\infty x e^{-(n+1)x} \; dx \] \[ I = \sum_{n=0}^\infty \int_0^\infty x d(\frac{e^{-(n+1)x}}{-(n+1)}) = \sum_{n=0}^\infty \Biggl( x \frac{e^{-(n+1)x}}{-(n+1)} \Biggr|_0^\infty - \int_0^\infty \frac{e^{-(n+1)x}}{-(n+1)} \; dx \Biggr) \] \[ I = \sum_{n=0}^\infty \Biggl( (0 - 0) + \frac{1}{n+1} \int_0^\infty e^{-(n+1)x} \; dx \Biggr) \] \[ I = \sum_{n=0}^\infty \frac{1}{n+1} \int_0^\infty e^{-(n+1)x} \; dx = \sum_{n=0}^\infty \frac{1}{-(n+1)^2} e^{-(n+1)x} \Biggr|_0^\infty \] \[ I = \sum_{n=0}^\infty \frac{1}{-(n+1)^2} (0-1) = \sum_{n=0}^\infty \frac{1}{(n+1)^2} \] \[ I = \sum_{n=1}^\infty \frac{1}{n^2} \] Now in order to evaluate this sum, we use the infinite sum of reciprocal squares which is the solution to the Basel problem: \[ \sum_{n=1}^\infty \frac{1}{n^2} = 1 + \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + ... = \frac{\pi^2}{6} \] \[ I = \frac{\pi^2}{6} \]
