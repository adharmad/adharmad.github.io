Integrate \( \int \sqrt{\frac{e^x-1}{e^x+1}} \; dx \)

\[ I = \int \sqrt{\frac{e^x-1}{e^x+1}} \; dx \] Substituting \( \frac{e^x-1}{e^x+1} = t^2 \Rightarrow e^x = \frac{1+t^2}{1-t^2} \Rightarrow dx = \frac{4t}{(1-t^2)(1+t^2)} \; dt \): \[ I = \int \sqrt{t^2} \frac{4t}{(1-t^2)(1+t^2)} \; dt = \int \frac{4t^2}{(1-t^2)(1+t^2)} \; dt \] \[ I = \int ( \frac{2}{1-t^2} - \frac{2}{1+t^2} ) \; dt = \int ( \frac{2}{(1-t)(1+t)} - \frac{2}{1+t^2} ) \; dt \] \[ I = \int ( \frac{1}{1-t} + \frac{1}{1+t} - \frac{2}{1+t^2} ) \; dt \] \[ I = -\ln \left| 1-t \right| + \ln \left| 1+t \right| - \tan^{-1}t + c \] \[ I = \ln \Biggl| \frac{1+t}{1-t} \Biggr| - \tan^{-1}t + c \] \[ I = \ln \Biggl| \frac{1+\sqrt{\frac{e^x-1}{e^x+1}}}{1-\sqrt{\frac{e^x-1}{e^x+1}}} \Biggr| - \tan^{-1}(\sqrt{\frac{e^x-1}{e^x+1}}) + c \]
